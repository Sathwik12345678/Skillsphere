# SkillSphere

SkillSphere is a freelance marketplace starter built with a React frontend and a Node/Express backend.

## Structure

- `client/` React + Vite + Tailwind frontend
- `server/` Node + Express + MongoDB backend
- `docs/` project documentation

## Run

### Backend

```bash
cd server
cp .env.example .env
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```
